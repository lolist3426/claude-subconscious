## ✨ What this is

claude-subconscious is a desktop app that gives Claude Code a “subconscious” layer for saved context, notes, and recall. It helps you keep track of past work, so Claude can stay aligned with your goals across sessions.

Use it when you want Claude to remember project details, personal rules, or ongoing tasks without having to repeat yourself each time.

## 💻 What you need

Before you install, make sure your Windows PC has:

- Windows 10 or Windows 11
- A stable internet connection
- At least 200 MB of free disk space
- Permission to run downloaded apps
- Claude Code installed and ready to use

If you plan to store a lot of notes, more free space will help.

## 🗂️ How it works

claude-subconscious keeps a local set of memory files that you can update over time. When you use Claude Code, the app helps surface the right context so your assistant can stay on task.

Common uses:

- Save project goals
- Keep coding rules in one place
- Store long-term task notes
- Track recurring prompts
- Reuse context across sessions

This can save time if you work on the same project over many days.
